CS170_Eight_Puzzle_Project1

This project was created to implement A* search to solve the eight-puzzle problem. We will be solving this problem using Uniform Cost Search, A* with Misplaced Tile heuristic and A\* with the Euclidean Distance heuristic.

This project was coded in C++. In our src folder you will find our implementation of the Node class, Problem Class and main folder.
